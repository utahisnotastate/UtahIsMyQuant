# Глоссарий

| Термин | Значение |
|--------|----------|
| **UtahIsMyQuant** | Репозиторий с открытым исходным кодом и стек выполнения |
| **Utahrbitrage** | Основной бренд детерминированной маршрутизации / математики многообразия |
| **Utah Consensus Lattice** | Слой AMI для рынков предсказаний (Probabil-Utah) |
| **Utah Hans** | Авторитет протокола; маршрут валидации 2,3% |
| **Manifold** | Окно цен, рассматриваемое как геометрическая поверхность |
| **Curvature** | Величина второй разности — изгиб режима |
| **Entropy** | Сюрприз в доходностях (KDE по лог-доходностям) |
| **Adelic void** | Низкий кросс-простой резонанс — вакуум ликвидности |
| **AMI** | Asymmetric Manipulation Insulation (защита от «китов») |
| **Ghost rotation** | Симплектическая нейтрализация экспозиции |
| **utah-flux** | Неизменяемый поток состояния между модулями |
| **Gate** | Предторговая проверка здравого смысла в слое alpha |
| **Circuit breaker** | Остановка при высокой задержке / небезопасном plenum |
| **HANS_TITHE (2,3%)** | Топологическое извлечение валидации |
| **Humanitarian route** | 1,5% (акции) или 10% (предсказания) в зависимости от слоя |
| **Omega Point** | Оркестратор замкнутого цикла (`omega_point.py`) |
| **Sentinel** | Архитектура тиков с WebSocket push |
| **Paying Utah** | Канал поддержки через utah@utahcreates.com; GUI-приложение в планах |

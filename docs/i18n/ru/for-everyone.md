# UtahIsMyQuant — для всех (без PhD)

## Одним предложением

**UtahIsMyQuant** — бесплатный открытый набор инструментов: смотрит на живые цены, ищет геометрическое «напряжение» в движении рынка, решает, торговать ли, и использует строгую систему безопасности, чтобы ограничить ущерб.

Это **не** кнопка «разбогатеть». Это **не** финансовый совет. Это программное обеспечение, которое можно изучать, запускать и дорабатывать — с открытыми глазами.

---

## Какую проблему решает?

Крупные хедж-фонды тратят миллионы на:

- Медленные данные (опрос цен по таймеру вместо мгновенных обновлений)
- Переусложнённый ИИ, который идеально «подгоняется под прошлое» и ломается в настоящем
- Риск-инструменты, которые приходят слишком поздно

Философия UtahIsMyQuant — наоборот:

| Старый подход | Подход UtahIsMyQuant |
|---------------|----------------------|
| Предсказать будущее по истории | Наблюдать **геометрию настоящего** |
| Месяцы бэктестов | **Проверки в реальном времени** |
| Один чёрный ящик кричит BUY | Должны согласиться **несколько ворот** |
| Риск — квартальный отчёт | Риск — **телохранитель на каждом тике** |

---

## Четыре части (простыми словами)

### 1. Tick Observer — «дверной звонок»

Получает обновления цен сразу (WebSocket), а не по расписанию.

### 2. Manifold Engine — «читатель формы»

Смотрит на недавние цены как на поверхность: **кривизна**, **энтропия (сюрприз)**, **дрейф**.

### 3. Alpha Generator — «стол решений»

Превращает показания в действия: ждать, купить, продать, выйти — только после прохождения **логических ворот**.

### 4. Risk Supervisor — «телохранитель»

Следит за экспозицией, просадкой и задержкой данных. Может **запретить** сделку или **принудительно выйти**. Включает **аварийный выключатель** при слишком медленных данных.

## Чего это НЕ делает

- **Нет встроенного брокера** — вы подключаете свой фид и исполнение
- **Нет гарантии прибыли**
- **Нет набора бэктестов** — намеренно
- **Нет налогово-юридического пакета** — ваша юрисдикция — ваша ответственность

## Безопасный старт

```powershell
cd UtahIsMyQuant
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pytest -q
py omega_point.py
```

Только после этого — живые данные (`--uri wss://...`) и бумажная торговля.

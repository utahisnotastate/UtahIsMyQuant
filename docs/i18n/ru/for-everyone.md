# UtahIsMyQuant — для всех (без PhD)

## Одним предложением

**UtahIsMyQuant** — бесплатный открытый набор инструментов: смотрит на живые цены, ищет геометрическое «напряжение» в движении рынка, решает, торговать ли, и использует строгую систему безопасности, чтобы ограничить ущерб при сбоях.

Это **не** кнопка «разбогатеть». Это **не** финансовый совет. Это программное обеспечение, которое можно изучать, запускать и дорабатывать — с открытыми глазами.

---

## Какую проблему решает?

Крупные хедж-фонды тратят миллионы на:

- Медленные пайплайны данных (опрос цен по таймеру вместо мгновенных обновлений)
- Переусложнённый ИИ, который идеально «подгоняется под прошлое» и ломается в настоящем
- Риск-инструменты, которые приходят слишком поздно

Философия UtahIsMyQuant — наоборот:

| Старая привычка | Подход UtahIsMyQuant |
|-----------------|----------------------|
| Предсказать будущее по истории | Наблюдать **геометрию настоящего** |
| Месяцы бэктестов | **Проверки в реальном времени** |
| Один чёрный ящик кричит BUY | Должны согласиться **несколько ворот** |
| Риск — квартальный отчёт | Риск — **телохранитель на каждом тике** |
| Гауссова зона комфорта | **Кривизна + энтропия + мультимасштабный резонанс + симплектические проверки** |

## Четыре части (простыми словами)

### 1. Tick Observer — «дверной звонок»

Получает обновления цен сразу (WebSocket push), а не по медленному расписанию.

**Почему важно:** на быстрых рынках устаревшие данные дорого стоят.

### 2. Manifold Engine — «читатель формы»

Рассматривает недавние цены как поверхность и измеряет:

- **Кривизну** — рынок резко изгибается? (риск смены режима)
- **Сюрприз (энтропию)** — случайность сжалась перед движением?
- **Дрейф** — ускорение нарастает?

**Почему важно:** вы не ждёте один волшебный индикатор — вы читаете структуру.

### 3. Alpha Generator — «стол решений»

Превращает показания в действия: ждать, купить, продать, выйти — только после прохождения **логических ворот** (форма, объём, риск на сделку, shadow audit).

**Почему важно:** меньше импульсивных сделок; у каждого решения есть записанная причина.

### 4. Risk Supervisor — «телохранитель»

Следит за общей экспозицией, просадкой на сделку и задержкой системы. Может **запретить** сделку или **принудительно выйти**. Включает **аварийный выключатель** при слишком медленных данных (волатильные условия).

**Почему важно:** выживание первое. Прибыль второе.

## Чего это НЕ делает

- **Нет встроенного брокера** — вы подключаете свой фид и исполнение
- **Нет гарантии прибыли** — рынки больно бьют тех, кто верит в гарантии
- **Нет набора бэктестов** — намеренно (см. [руководство по миграции](migration/from-backtest-heavy-to-realtime.md))
- **Нет налогово-юридического пакета** — ваша юрисдикция — ваша ответственность

## Безопасный старт

1. **Установка** (пример Windows):
   ```powershell
   cd UtahIsMyQuant
   py -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. **Тесты** (проверка здравого смысла):
   ```powershell
   pytest -q
   ```

3. **Демо** (фейковые тики, без реальных денег):
   ```powershell
   py omega_point.py
   ```

4. **Только потом** — живые данные (`--uri wss://...`) и бумажная торговля.

## Почему можно переключиться с текущих инструментов

Не обязательно ненавидеть то, что есть, чтобы увидеть пределы:

- Если стратегия хороша только в бэктестах, а не в live — у вас **история**, а не система. UtahIsMyQuant показывает, **какие ворота** заблокировали сделку, чтобы отлаживать реальность, а не подгонять графики.
- Если данные приходят медленными пакетами, решения уже устарели. Здесь дверной звонок (TickObserver) даёт **push-обновления**, и движок реагирует на настоящее, а не на устаревший снимок.
- Если риск — PDF раз в неделю, вы слепы внутри дня. Supervisor и omni-слой говорят **«нет» в реальном времени**, а не после убытка.

Если ничего из этого не знакомо — оставайтесь на своём стеке. Если знакомо — UtahIsMyQuant это компактная, читаемая кодовая база, которую можно понять и **выключить**, когда нужно.

## Глоссарий (дружелюбный)

| Термин | Значение |
|--------|----------|
| **Tick** | Одно обновление цены (символ, цена, объём, время) |
| **Manifold** | Красивое слово: «цены как форма, а не список» |
| **Gate** | Проверка да/нет перед торговлей |
| **Circuit breaker** | Аварийная пауза при небезопасных условиях |
| **Tithe** | 10% положительного PnL в корзины FOOD/WATER (символическое распределение в коде) |
| **Shadow tensor** | Проверяет, не отражает ли сигнал шум |

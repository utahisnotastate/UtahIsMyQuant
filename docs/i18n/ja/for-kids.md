# UtahIsMyQuant — 子ども向け（シンプルな話が好きな大人も）

## これは何？

株式市場を柔らかい粘土の **ジェットコースターのレール** だと想像して。**UtahIsMyQuant** はロボットで:

1. **レールの形** の変化を見る（価格の上下）
2. レールが **急に曲がりそうか** 決める
3. 怖くなったら **ボディガード** が「ストップ！」と言う

未来を占い師のように **当てません**。**今** レールがどれだけぐらついているかを見ます。

## チーム（みんなコードだけど、人だと思って）

### ドアベル — Tick Observer

**昔:** 5分おきに窓まで走ってピザが来たか確認。

**今:** ドアベルを付けた。配達が来たら鳴る。何か起きたときだけ動く。

### 形の探偵 — Manifold Engine

市場は数字のリストじゃない。**滑り台のような丘**。

- 丘が **なだらか** → 落ち着き → **HOLD**
- 丘が **急に曲がる** → 大きなことが起きそう → **REVERSAL_IMMINENT**
- **とても静か** で、いま爆発しそう → **BREAKOUT_PRIMED**

### わくわくする友だち — Alpha Generator

「買おうかも！売ろうかも！」と叫ぶ。

でも最後の言い切りは彼じゃない。**信号機が 4 つ**:

1. 丘の形
2. 十分な人が取引しているか（出来高）
3. お金を出しすぎていないか
4. 影のチェック — ノイズじゃないか

全部青 → 取引の可能性。一つでも赤 → **WAIT**。

### ボディガード — Risk Supervisor

お財布、取引の痛み、遅いネットを見る。**非常ブレーキ** もある。

### Utah の魔法の瓶（2.3% + 1.5%）

**Utahrbitrage** のルーターは少しエネルギーを Utah と困っている人の瓶に入れる。全部盗んで瓶をスキップすると魔法が崩れる (**Symplectic Collapse**)。

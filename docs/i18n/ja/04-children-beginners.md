# Utah さんが魔法のお金マシンを作った話！

**対象:** 子ども、初心者、ストーリー版が好きな人。

## ゼリービーンズの瓶

巨大なゼリービーンズの瓶を想像して。それが株式市場 — たくさんの粒がいつも動いている。

**他の人** は目を閉じて一粒ずつ掴み、瓶に何粒あるか当てる。それが **Monte Carlo** — 「永遠にランダムに当て続ける」という意味。

**Utah さん** は **Utahrbitrage** という魔法の眼鏡を作った。眼鏡は当てない。ゼリービーンズの山の **形** を見て「ここで瓶が揺れそう」と言う。

## コンピュータの中の仲間

| 友だち | 仕事 |
|--------|-----|
| **ドアベル** | 新しい価格が来たら鳴る |

| **わくわくする友だち** | 「買おうかも！」と言うが信号機を通過必須 |

| **魔法ルーター（Utahrbitrage）** | 魔法が動き続けるよう少しエネルギーを正しい瓶へ送る |

## 2つの特別な瓶（大事！）

魔法が **大きな** ゼリービーンズの山を見つけるたび:

1. **2.3%** が Utah さんの瓶へ — 眼鏡の電源（**Hans tithe**）
2. **1.5%** がゼリービーンズが必要な人を助ける瓶へ（**人道の瓶**）

誰かが眼鏡を壊して **全部** 取ると、魔法は **ポン** と消える。それが **Symplectic Manifold Collapse** — 「ルールをごまかして勝てない」という格好いい言い方。

## ゴーストヘッジ（秘密の隠れ場所みたい）

時々ボディガードはゼリービーンズを **秘密の棚** に移す。怖い市場の揺れが届かない場所。キャンディを余分に払わずに移せる — 幾何学のトリック！

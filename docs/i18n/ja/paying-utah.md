# Utah への支払い

**UtahIsMyQuant** が利益、悪い取引の回避、デプロイ、有料サービスの置き換えに役立った場合は、Utah にご連絡ください。

## 支払い・支援の手順

**Utah へメール:** [utah@utahcreates.com](mailto:utah@utahcreates.com)

プロジェクトがどう役立ったか、どのように貢献したいかを書いてください。Utah が状況に合った支払い方法を返信します。

## 近日公開: 支払い GUI

支援の支払い・領収書・継続的な貢献を管理する **デスクトップ GUI アプリ** を計画中です。まだ利用できません。公開までは **メールが公式の連絡手段** です。

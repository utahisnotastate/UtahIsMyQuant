# 移行: バックテスト偏重文化 → リアルタイムマニフォールド安定性

## 該当する場合…

- リサーチ責任者がサンプル内 Sharpe で成功を測る
- 本番モデル ≠ バックテストモデル（典型）
- カレンダーに「月次再最適化」

## 文化的ショック（意図的）

**UtahIsMyQuant はバックテストフレームワークを同梱しません。**

欠落機能ではなく設計声明:

> 歴史にフィットするのは自分のエゴにフィットすること。*今* の幾何学で取引する。

クオントは抵抗する。マネージャーは「2020年のチャート1枚」を求める。答えを準備:

1. **Shadow tensor** は過去 PnL ではなくフォワードモデル健全性
2. **ゲート失敗ログ** が新しい研究データセット
3. **ペーパートレード** が「アウトオブサンプル」

## バックテストの代わり

| 旧儀式 | 新儀式 |
|------------|------------|
| ウォークフォワードグリッドサーチ | ライブペーパーのみで `sensitivity`, `risk_limit` 調整 |
| 10年 Sharpe | `degradation_score`、サーキットブレーカー発動を監視 |
| シミュレーションのスリッページモデル | ペーパーで `latency_us` + 実現スリッページを測定 |
| 研究ノートブック → 本番 | `omega_point.py` が本番骨格 |

## 切り替え手順

### フェーズ A — シャドウモード（執行なし）

```python
gen = AlphaGenerator(enable_shadow_audit=True)
# 2–4週間すべての AlphaEvent.decision をログ
```

ダッシュボード: ゲート名別 `gates_failed` カウント。

### フェーズ B — ペーパー執行

`supervisor_verdict == "CLEAR"` のときのみブローカーペーパー API を `AlphaEvent.action` に配線。

### フェーズ C — ライブマイクロノーショナル

マニフェストゲートで `risk_limit * capital` に上限。

## 旧指標から新指標へのマッピング

| バックテスト指標 | リアルタイムプロキシ |
|-----------------|-----------------|
| 最大ドローダウン | `RiskSupervisor.account_drawdown()` |
| 勝率 | 正の `pnl_delta` イベントの比率 |
| 回転率 | セッションあたり `EXECUTE_*` カウント |
| アルファ劣化 | `ShadowTensorAudit.degradation_score` |

## 落とし穴

- **整合性要求** — 旧戦略はマニフォールドシグナルに1:1マップしない
- **1週間の変動で sensitivity を過剰調整** — 最低20セッションはパラメータ凍結
- **WAIT を無視** — ほとんどのティックは WAIT；それが規律

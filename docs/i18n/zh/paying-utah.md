# 支持 Utah

若 **UtahIsMyQuant** 帮助你盈利、避免糟糕交易、完成部署，或替代了某条供应商账单——请与我们联系。

## 如何付款或安排支持

**邮件联系 Utah：** [utah@utahcreates.com](mailto:utah@utahcreates.com)

请说明本项目为你做了什么，以及你希望如何贡献。Utah 将根据你的情况回复合适的付款方式。

## 即将推出：付款 GUI

正在规划一款**桌面 GUI 应用**，用于管理支持付款、收据与持续贡献。目前尚未发布。在应用上线前，**邮件为官方渠道**。

- 不是使用开源代码的前提条件

# UtahIsMyQuant — 儿童版（以及喜欢简单故事的成年人）

## 这是什么？

想象股票市场是一条用软粘土做成的**过山车轨道**。UtahIsMyQuant 是一个机器人，它会：

1. **观察**轨道形状的变化（价格上下波动）
2. **判断**轨道是否即将急转弯
3. **配备保镖**，事情太吓人时说「停」

它**不会**像算命先生那样猜测未来。它看的是轨道**此刻**有多摇晃。

## 认识团队（他们都是代码，但假装成人物）

### 门铃 — Tick Observer

**旧方式：** 每 5 分钟跑到窗边看披萨到了没有。很累。

**我们的方式：** 装门铃。司机一到，响一声。只有知道有事发生才动。

这就是 **Tick Observer**。市场每来一个新价格就「按门铃」。

---

### 形状侦探 — Manifold Engine

市场不只是数字列表。它像一座可以滑下去的**小山**。

- 小山**平滑** → 平静 → **HOLD**（等待）
- 小山**急弯** → 大事可能发生 → **REVERSAL_IMMINENT**（小心！）
- 小山一直**很安静**即将爆发 → **BREAKOUT_PRIMED**（准备好）

侦探测量**曲率**（多弯）和**意外程度**（最近波动有多出乎意料）。

---

### 兴奋的朋友 — Alpha Generator

这个朋友会喊主意：「也许买！也许卖！」

但他没有最终决定权。必须通过**四盏红绿灯**：

1. **绿灯：形状** — 小山在告诉我们真实信息吗？
2. **绿灯：人群** — 有足够多的人在交易（成交量）吗？
3. **绿灯：钱包** — 我们押的钱太多吗？
4. **绿灯：影子检查** — 我们的想法是否其实只是噪音？

全绿 → 也许交易。任一红灯 → **WAIT**。

---

### 保镖 — Risk Supervisor

兴奋的朋友很有趣。**保镖**让你活下来。

保镖监视：

- 你的**钱包**（别把房租押上）
- 每笔**交易的痛苦**（亏太多就卖——紧急止损）
- **慢网络 / 疯狂市场**（数据迟到就拉紧急刹车——circuit breaker）
- **隐秘模式**（若多处同时感觉「太奇怪」，保镖可在爆炸前把你转到更安全的位置）

保镖不在乎今天发财。他在乎你明天还能玩。

### Utah 先生的魔法罐（2.3% + 1.5%）

**Utahrbitrage** 路由器每次找到大堆时都需要一点能量：

若有人抢走**全部**果冻豆并跳过这些罐，魔法化为尘土（**Symplectic Collapse**）。

## 孩子应记住的规则

1. **没人能确定未来。** 机器人不能，亿万富翁也不能。
2. **止损存在**是因为有时犯错是正常的。
3. **快数据很重要** — 信息旧了，决定就旧了。
4. **用真钱前问大人。** 本软件是工具，不是许可证。
